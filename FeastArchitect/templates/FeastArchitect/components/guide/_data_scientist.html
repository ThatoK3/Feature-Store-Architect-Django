Using the Search & Discovery Tools
🔍

Global Search

Search across feature names, descriptions, tags, and data sources. Use Ctrl+F to quickly find features by business context.

🏷️

Tag Filtering

Browse features by tags like #fraud, #recommendations, or #pii to find relevant feature groups.

🔗

Lineage View

Trace features back to source databases. Understand dependencies and upstream data quality issues.

📈

Statistics Bar

Click the stats bar to see repository overview. Check feature counts, data source coverage, and service utilization.

Experimentation with AI Assistant

Use the AI Assistant (Ctrl+Shift+A) to:

  • Generate Code: Auto-create feature definitions from descriptions
  • Explain Lineage: Natural language explanation of data flow
  • Suggest Features: Get recommendations based on your use case
  • Validate Design: Check for common anti-patterns
Example prompts:
• "Create features for churn prediction using user activity"
• "Explain the data lineage for the fraud detection service"
• "What features are missing for a recommendation system?"
• "Optimize these features for low-latency serving"
